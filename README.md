## Getting Started

### Prerequisites

Python 3.x

### Installing

Installing the requirements

```
pip install -r requirements.txt
```

## Running 

1. Clone the repo

```
git clone https://github.com/LamNg99/Vertebrae-Evaluation-Application.git
```

2. Run `python3 app.py`
